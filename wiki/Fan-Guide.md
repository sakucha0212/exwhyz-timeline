## 1. 画面の見かた

### 2つの画面モード

このサイトには **ハイライト** と **タイムライン** の2つの画面があります。
画面上部のタブで切り替えられます。

```
┌──────────────────────────────────────────────────────────┐
│                   ExWHYZ Timeline                        │
│                輝きの軌跡 × あなたの思い出                 │
├──────────────────────────────────────────────────────────┤
│   [ ハイライト ]  [ タイムライン (2024年5月) ]             │
├──────────────────────────────────────────────────────────┤
│                                                          │
│                    （各画面の内容）                         │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

### ハイライト画面

ExWHYZ の全イベントを **年ごと・カテゴリごと**に一覧できます。  
全体の歴史をざっくり振り返りたいときに便利です。

### タイムライン画面

選んだ月のイベントと X ポストを **日付順**に表示します。  
あの日の思い出をじっくり振り返りたいときに。

## 2. ハイライトでイベントを振り返る

### ハイライト画面の構成

```
┌──────────────────────────────────────────────────────────┐
│  [すべて(47)] [ツアー(9)] [ライブ(17)] [発表(9)]           │  ← フィルター
│  [イベント(2)] [リリース(10)]                              │
├──────────────────────────────────────────────────────────┤
│  ▼ 2026年 (15件)                                         │
│  │ ▌🎵 ExWHYZ LAST LIVE '光'           ライブ     │
│  │ ▌💿 3rdアルバム「zION」リリース       リリース  │
│  │ ▌🎵 LAST TOUR 'DANCE YOUR DANCE'     ツアー     │
│  │ ...                                                   │
│  ▼ 2025年 (8件)                                          │
│  │ ...                                                   │
│  ▼ 2024年                                                │
│  │ ...                                                   │
└──────────────────────────────────────────────────────────┘
```

### カテゴリで絞り込む

上部のフィルターボタンで、見たい種類のイベントだけを表示できます。

| ボタン | 表示されるイベント |
|--------|-------------------|
| すべて | 全イベント |
| ツアー | ツアー情報のみ |
| ライブ | ライブ・コンサートのみ |
| 発表 | 告知・発表のみ |
| イベント | その他イベントのみ |
| リリース | CD・配信リリースのみ |

- 複数のボタンを同時に選択できます（例: ツアー + ライブ）
- 「すべて」を押すと全件表示に戻ります

### 年の開閉

▶ マークをクリックすると、その年のイベント一覧を開いたり閉じたりできます。

### カードをクリックすると…

イベントカードをクリックすると、その月の**タイムライン画面**に自動で切り替わります。  
「あの時のポストを見たい！」と思ったら、ハイライトから目的のイベントを探してクリック！

## 3. タイムラインで月ごとに見る

### タイムライン画面の構成

```
┌──────────────────────────────────────────────────────────┐
│  ← 前月   [2024年 ▼] [5月 ▼]   次月 →   [🔄 更新]       │  ← 月ナビゲーション
├──────────────────────────────────────────────────────────┤
│  📅 2024年5月20日（月）                                    │  ← 日付見出し
│  ┌──────────────────────┬──────────────────────────────┐ │
│  │ 🎤 活動情報           │ 💬 X ポスト                  │ │
│  │ ライブ開催            │ （あなたのポストが表示）      │ │
│  └──────────────────────┴──────────────────────────────┘ │
│                                                          │
│  📅 2024年5月22日（水）                                    │
│  ┌──────────────────────┬──────────────────────────────┐ │
│  │ 📢 新曲リリース発表   │ 💬 X ポスト                  │ │
│  └──────────────────────┴──────────────────────────────┘ │
└──────────────────────────────────────────────────────────┘
```

### 活動情報カード（左カラム）

公式の活動情報が表示されます。カテゴリごとに色分けされています。

| カテゴリ | 内容 |
|---------|------|
| 🎤 ライブ | コンサート・ライブイベント |
| 💿 リリース | CD・配信リリース |
| 📺 メディア | テレビ・ラジオ・雑誌出演 |
| 📢 発表 | 新情報・告知 |

### X ポスト（右カラム）

あなたが X に投稿したポストが埋め込み表示されます。  
ポスト内の「Xで見る」リンクから X のページを開くこともできます。

> 📱 **スマホで見ている場合**: 活動情報 → X ポストの順に縦並びで表示されます。

### ハイライトに戻る

「ハイライト」タブをクリックすると、いつでもハイライト画面に戻れます。

## 4. X（Twitter）のポストを表示する

X ポストを表示するには、**X アカウントでのログインが必要**です。

ログインすると、あなたが X に投稿したポストの中から ExWHYZ に関連するものが自動で検索・表示されます。

### 表示されるポストの条件

以下のキーワードを含む、**あなた自身のポスト**が対象です（リツイート・リプライは除外）。

- グループ名: `ExWHYZ` / `WHYZ` / `イクスワイズ` / `イクス` など
- メンバー名: `mikina` / `みきな` / `ミキナ` / `yu-ki` / `ゆーき` / `maho` / `まほ` / `mayu` / `まゆ` / `now` / `なう` / `midoriko` / `みどりこ` など
- 関連: `EMPiRE` / `俺とお前で音源チェック` など

> 💡 **ヒント**: 当時のポストが表示されない場合は、上記のキーワードがポストに含まれているか確認してみてください。

## 5. ログインする方法

### 手順

1. サイトにアクセスすると、**ログイン画面**が表示されます
2. **「X でログイン」** ボタンをクリック
3. X の認証画面が開くので、**「アプリを認証」** をクリック
4. ログイン完了！タイムラインが表示されます

### ログインに必要なもの

- X（Twitter）アカウント

### プライバシーについて

- このアプリはあなたの **公開ポストのみ**を読み取ります
- ポストの投稿・削除などの操作は行いません
- 取得したデータはあなたの**ブラウザ内にのみ保存**されます
- サーバーにポストデータは保存されません

## 6. ポストを最新の状態に更新する

**当月を表示しているとき**、タイムライン画面の上部に **「🔄 更新」ボタン**が表示されます。

このボタンをクリックすると、前回の取得以降に投稿した新しいポストを追加で読み込みます。

### 更新できないとき

「現在ポストを更新できません。」というメッセージが表示された場合、X API の利用制限に達しています。  
しばらく時間をおいてから再度お試しください。

> ⏰ **目安**: 制限は一定時間後に自動でリセットされます。

## 7. よくある質問（FAQ）

### Q. ポストが表示されない

**A.** 以下を確認してください：

1. **ログインしているか** — ログインしていないとポストは表示されません
2. **キーワードが含まれているか** — [表示されるポストの条件](#表示されるポストの条件) を確認してください
3. **正しい月を選んでいるか** — ポストを投稿した月に移動してみてください

---

### Q. ハイライト画面の「タイムライン」タブが押せない

**A.** タイムラインタブは、ハイライト画面でイベントカードをクリックして月を選択すると有効になります。  
まずハイライトから見たいイベントを探してクリックしてください。

---

### Q. 活動情報が少ない・間違っている

**A.** 活動情報は手動で管理されているデータです。  
不足・誤りを見つけた場合は、GitHub の Issues からご報告いただけると助かります。

---

### Q. スマホで見ると表示が崩れる

**A.** スマホでは1カラム表示（縦並び）になります。これは正常な動作です。  
横向きにすると2カラム表示に近い見た目になる場合があります。

---

### Q. 過去のポストが全部表示されない

**A.** 初めてログインした月は、2022年6月から現在までの全ポストを取得します。  
ポスト数が多い場合、取得に時間がかかることがあります。  
また、X API の利用制限により一度に取得できる件数に上限があります。

---

### Q. ログアウトしたい

**A.** 現在のバージョンではログアウト機能は実装されていません。  
ブラウザのキャッシュ・Cookie をクリアすることでログアウト状態になります。
