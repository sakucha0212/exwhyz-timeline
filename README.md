# ExWHYZ Timeline

> ExWHYZ の奇跡と、あなたの思い出をつなぐアーカイブファンサイト

```
📅 2024年5月20日
┌─────────────────────────────┬─────────────────────────────┐
│ 🎤 ExWHYZ 公式活動情報       │ 💬 あなたの X ポスト         │
│                             │                             │
│ ライブ「〇〇 TOUR」開催      │ 「最高だった！！！」         │
│ 会場: 〇〇ホール             │ 「みきなのソロパートで泣いた」│
└─────────────────────────────┴─────────────────────────────┘
```

---

## 🎯 概要

日本のダンス＆ボーカルグループ **ExWHYZ** の公式活動記録と、ファンが X（Twitter）に投稿したリアルな思い出を日付ごとにシンクロさせて振り返ることができるファンサイトです。

- 📅 **2022年6月〜2026年9月**の活動を月単位でアーカイブ
- 🎤 ライブ・リリース・メディア出演などの公式活動情報を表示
- 💬 X（Twitter）にログインすると、自分のポストを活動記録と並べて見返せる
- 📱 スマホファーストのレスポンシブデザイン（ダークモード + ピンクアクセント）

---

## 🚀 クイックスタート

```bash
cd exwhyz-timeline
npm install
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

> 初回起動は `.env.local` の作成が必要です。詳細は [Developer Guide](wiki/Developer-Guide.md) を参照してください。

---

## 🛠️ 技術スタック

- **Next.js 16** (App Router) + **TypeScript**
- **Tailwind CSS** — スタイリング
- **NextAuth.js** — X OAuth 2.0 認証
- **twitter-api-v2** — X API クライアント
- **IndexedDB** — 月単位ツイートキャッシュ
