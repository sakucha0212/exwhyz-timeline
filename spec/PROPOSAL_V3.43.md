# ExWHYZ-Timeline ハイライトカード左ボーダー追加 変更要求書（v3.43）

## 📋 変更要件の整理

### v3.43での変更要件

1. **UI改善**: ハイライトカードの左側にタイプ別の色付きボーダーを追加し、タイプを視覚的に判別しやすくする

---

## 1. 現状（v3.42）と問題の詳細

### 問題

ハイライトカードのタイプ（ツアー/ライブ/フェス/発表）は右上の小さなバッジのみで判別する必要があり、スクロール中など一覧を見渡す際にタイプがわかりづらい。

### 改善案

年ヘッダーと同様に、カードの左側に `border-l-4` の色付きボーダーを追加する。

```
┌──────────────────────────┐
│▌🎵 LAST TOUR 'DANCE...  ツアー│
└──────────────────────────┘
```

---

## 2. 変更詳細

### 2.1 `components/Highlights/HighlightCard.tsx`（修正）

- `getTypeBorderColor` 関数を追加
- カード div に `border-l-4` + 動的ボーダー色クラスを追加

| タイプ | ボーダー色 |
|--------|-----------|
| ツアー | `border-l-pink-500/60` |
| ライブ | `border-l-green-500/60` |
| フェス | `border-l-orange-500/60` |
| 発表 | `border-l-blue-500/60` |
