# ExWHYZ-Timeline 年アコーディオン内に時系列矢印ライン追加 変更要求書（v3.44）

## 📋 変更要件の整理

### v3.44での変更要件

1. **UI改善**: 年アコーディオン内の左端に縦線＋下矢印のタイムライン表示を追加し、時系列の流れを視覚化する

---

## 1. 現状（v3.43）と問題の詳細

### 問題

年アコーディオン内にカードがフラットに並んでおり、時系列（降順：最新が上）の流れが視覚的にわかりづらい。

### 改善後

```
▼ 2026年 (15件)
┌──────────────────────────────┐
│ │ ▌🎵 LAST TOUR...    ツアー │
│ │ ▌🎵 CLUB Ex...      ライブ │
│ │ ▌🎵 PEDROWHYZ...    ライブ │
│ ↓ ▌🎵 TIF 2026...    フェス │  ← 下端に下矢印
└──────────────────────────────┘
```

左端に縦線（`border-l-2 border-gray-600`）を引き、最下部に `↓` 矢印を配置する。
