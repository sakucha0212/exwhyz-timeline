# ExWHYZ-Timeline ハイライトのタイムライン表示改善（単純な上下線のみ）変更要求書（v3.46）

## 📋 変更要件の整理

### v3.46での変更要件

1. **UI改善**: 年アコーディオン内のタイムライン表示を、下矢印から単純な縦線のみに変更する

---

## 1. 現状（v3.45）と問題の詳細

### 問題

現在のタイムライン表示は「左端の縦線＋下端の下矢印SVG」だが、矢印が下部だけにありカードとのつながりが弱い。また、矢印が目立ちすぎる。

### 改善後

```
▼ 2026年 (15件)
┌──────────────────────────────┐
│ │                            │
│ │ ▌🎵 LAST TOUR...    ツアー │  ← 左端に縦線のみ
│ │                            │
│ │ ▌🎵 CLUB Ex...      ライブ │
│ │                            │
│ │ ▌🎵 PEDROWHYZ...    ライブ │
│ │                            │
└──────────────────────────────┘
```

| 要素 | 内容 |
|------|------|
| 縦線 | 上から下まで単純な縦線 `border-l-2 border-gray-600` |
| 矢印 | 削除 |
| ● | 不要（追加しない） |

---

## 2. 変更詳細

### 2.1 `components/Highlights/YearSection.tsx`（修正）

- 下矢印 SVG を削除
- ● ポイントは追加しない
- 左端に単純な縦線のみを表示（既存の縦線は維持）
