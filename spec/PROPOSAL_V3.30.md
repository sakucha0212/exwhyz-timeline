# ExWHYZ-Timeline EventCard バッジ位置・日付フォーマット修正 変更要求書（v3.30）

## 📋 変更要件の整理

### v3.30での変更要件

1. **UI修正**: カテゴリバッジを右端に移動
2. **UI修正**: 日付表示をタイトルの下に移動
3. **バグ修正**: ツアー期間の日付フォーマットを `yyyy年MM月dd日` に統一

---

## 1. 現状（v3.29）と問題の詳細

### 問題

- カテゴリバッジがタイトル行の下に左寄せで表示されており、右側の余白が活用されていない
- ツアー期間の日付が `5/6〜7/4` のような省略形式で表示されており、月日だけでは年がわからない

### 改善後のイメージ

```
┌──────────────────────────────────────────┐
│ 🎵 LAST TOUR 'DANCE YOUR DANCE'  🎵 ライブ│  ← 左=タイトル、右=カテゴリバッジ
│  2026年5月6日 〜 2026年7月4日            │  ← 日付はタイトルの下に
└──────────────────────────────────────────┘
```

---

## 2. 変更詳細

### 2.1 `components/Overview/EventCard.tsx`（修正）

| 項目 | 変更前 | 変更後 |
|------|--------|--------|
| トップ行レイアウト | 左=タイトル、右=日付 | 左=タイトル、**右=カテゴリバッジ** |
| 日付表示位置 | トップ行の右側 | タイトル行の**下** |
| 日付フォーマット（単発） | `2026年8月31日` | `2026年8月31日`（変更なし） |
| 日付フォーマット（ツアー） | `5/6〜7/4` | `2026年5月6日 〜 2026年7月4日` |
| カテゴリバッジ位置 | 日付下に左寄せ (`mt-1`) | **トップ行の右端** |
| `formatDateShort` 関数 | 使用 | **削除**（不要） |
