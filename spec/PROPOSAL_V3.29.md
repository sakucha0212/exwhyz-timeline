# ExWHYZ-Timeline EventCard レイアウト改善 変更要求書（v3.29）

## 📋 変更要件の整理

### v3.29での変更要件

1. **UI改善**: EventCard のレイアウトを見直し、右側の無駄な余白を解消してコンパクトにする

---

## 1. 現状（v3.28）と問題の詳細

### 問題

現在の EventCard はタイトル、日付、カテゴリバッジがすべて左寄せで縦積みされており、右側がスカスカで無駄な余白が目立つ。

- タイトル行の右半分が空いている
- カード内の `p-4` + `ml-7` インデントで縦に長い
- カード間隔 `space-y-3` もやや広め

### 改善方針

- トップ行を `flex justify-between` で左右分割（左=タイトル、右=日付）
- パディングを `p-4` → `p-3` に縮小
- カード間隔を `space-y-3` → `space-y-2` に縮小
- `ml-7` インデントを不要に

### 改善後のイメージ

```
┌──────────────────────────────────┐
│ 🎵 LAST TOUR 'DANCE...  5/6〜7/4 │  ← flex justify-between
│ 🎵 ライブ                        │  ← カテゴリバッジ
└──────────────────────────────────┘
```

## 2. 変更詳細

### 2.1 `components/Overview/EventCard.tsx`（修正）

| 項目 | 変更前 | 変更後 |
|------|--------|--------|
| カードパディング | `p-4` | `p-3` |
| トップ行レイアウト | タイトルのみ左寄せ | `flex justify-between items-start`（左=タイトル、右=日付） |
| 日付表示位置 | タイトル下に `ml-7` インデント | タイトルと同じ行の右側、`text-right whitespace-nowrap` |
| カテゴリバッジ | 日付下に `ml-7 mt-2` | タイトル行の下に `mt-1`（インデントなし） |

| 項目 | 変更前 | 変更後 |
|------|--------|--------|
| カード間隔 | `space-y-3` | `space-y-2` |
