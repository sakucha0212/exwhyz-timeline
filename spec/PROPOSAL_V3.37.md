# ExWHYZ-Timeline MonthPagination レイアウト ロールバック 変更要求書（v3.37）

## 📋 変更要件の整理

### v3.37での変更要件

1. **UI修正**: `MonthPagination` のレイアウトを v3.33 以前の従来型（前月→年月→次月）に戻す

---

## 1. 現状（v3.36）と問題の詳細

### 問題

v3.34 で案A（左右分離型）を適用したが、前後移動と年月指定が分離しすぎて、むしろ分かりづらくなった。

```
現状: [◀ 前月] [次月 ▶]    [2026年▼] [7月▼]    空の日を非表示  読み込み中...  [更新]
        ← 連続移動 →         ← 直接指定 →         ← その他オプション →
```

### 改善方針

v3.33 以前のレイアウトに戻す。矢印ボタンで年月を挟み込む最も一般的なパターン。

---

## 2. 変更詳細

### 2.1 `components/Timeline/MonthPagination.tsx`（修正）

v3.34 で追加したグループ化 div を削除し、元のフラットな並びに戻す。

**変更後:**
```
[◀ 前月] [2026年▼] [7月▼] [次月 ▶]     空の日を非表示  読み込み中...  [更新]
```

| 変更点 | 内容 |
|--------|------|
| 前月/次月グループ div | 削除 |
| 年月ドロップダウングループ div | 削除 |
| 要素の並び順 | 前月 → 年ドロップダウン → 月ドロップダウン → 次月（v3.33以前と同じ） |
